# DEMiCo
DEMiCo: A Music-Induced Emotion Dataset Integrating Physiological Signals, Dual Emotion Models and Listener Profiles

This anonymized repository provides review-time resources associated with the submitted paper. 
Due to double-anonymous review and copyright/privacy considerations, the full dataset release is currently under preparation. 
This repository includes the dataset structure, field descriptions, example files, and scripts/templates for reproducibility. 
The complete release will include anonymized metadata, music links with timestamps, emotion annotations, physiological features, and listener-level profile variables, subject to ethical and copyright constraints.
